Paginator

class Paginator extends Plugin implements AdaptCollectionQuery, FormatCollection, ResponseHeaders

A basic paginator that takes limit and page from the request and adapts the colleciton queries accordingly.

By default, the plugin adds http headers with meta info to the response: X-Pagination-Total, X-Pagination-Limit and X-Pagination-Page

These can be renamed or moved to the response payload instead, through the plugin's configuration options.
