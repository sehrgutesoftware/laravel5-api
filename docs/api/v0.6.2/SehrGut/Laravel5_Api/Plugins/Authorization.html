protected User authenticatedUserOrDummy()

Laravel assumes an authenticated User object for any authorization.

There might be situations where you don't have or need an User in your authorisation logic. For those situations we will simply create a DummyUser which contains no information in order to satisfy Laravel's authorisation logic.
